1.ユーザーとの関係

1.1 ユーザーは Twitter Search の日付指定式ツイート検索サービス（以下、本文書で総して「サービス」といいます）をご利用になる場合、本規約に定められている条件が適用されます。

1.2 Twitter Search はユーザーが本規約の範囲内で十分なサービスを受けられることを前提に条件を定めています。

2.ユーザータイムラインの利用

2.1 本サービスは、ユーザーが検索条件に設定したツイッターユーザーのタイムラインを保護されたツイッターユーザーでない限り、取得・表示できるものとします。ただし、Twitter の利用規約を超えて、これらを利用することはありません。

2.2 ユーザーは、Twitter および本サービスの利用規約にしたがってユーザータイムラインを検索・閲覧できることといたします。

2.3 ユーザーは、検索対象のツイートから得られた情報を、自由に利用できるとは限りません。

2.4 ユーザーは、ユーザータイムラインの検索・閲覧のみ可能とし、改変などの行為はできないものとします。

3.ユーザータイムラインの管理

3.1 本サービスは、Twitter API 消費削減のためにキャッシュを設けています。このため、本サービスはユーザータイムラインをキャッシュ内で一時的に管理できるものとします。ただし、これらのデータは改変されず、安全に管理されます。

3.2 ユーザーは、本サービスが設けているキャッシュにアクセスすることができないものとします。

3.3 ユーザーは、本サービスがユーザータイムラインをキャッシュ内で管理している限り、最新のツイートを入手できないものとします。

4.プライバシーと個人情報

　Twitter Search は、個人情報の保護についての趣旨をプライバシーポリシーで規定しております。次の文書を参照してください。　プライバシーポリシー

5.本規約の変更

5.1 本規約は、Twitter Search が変更し、この報告は本ウェブサイト上にて掲載いたします。

5.2 本規約の変更後、ユーザーはこの規約が適用されることといたします。
